# Toffee Auto Playlist

Automatic M3U/JSON playlist for Toffee Live channels with periodic updates.

### About

This repository provides an always-updated playlist file for Toffee Live (Bangladesh) channels in both JSON and M3U format.  
It includes an introductory credit video at the beginning of the playlist.

**Main playlist links** (always latest version):

- JSON: https://raw.githubusercontent.com/johirxofficial/Toffee-Auto-Playlist/main/toffee_playlist.json  
- M3U:  https://raw.githubusercontent.com/johirxofficial/Toffee-Auto-Playlist/main/toffee_playlist.m3u

### How to Use – Android

1. **Recommended Players** (free & paid both work well):
   - IPTV Smarters Pro
   - Tivimate (highly recommended)
   - GSE Smart IPTV
   - Perfect Player
   - OTT Navigator

2. **Steps**:
   - Open your chosen IPTV player app
   - Go to **Add Playlist** / **New Playlist** / **M3U URL**
   - Paste this link:  
     `https://raw.githubusercontent.com/johirxofficial/Toffee-Auto-Playlist/main/toffee_playlist.m3u`
   - Give it a name (e.g. "Toffee Live")
   - Save / Load playlist
   - Channels should appear (may take 10–30 seconds depending on app & internet)

3. **Tips**:
   - Use a good VPN if channels buffer or show geo-error
   - Clear app cache if channels don't load after update
   - Update playlist manually every 2–3 days (most apps have auto-refresh option)

### How to Use – Desktop / PC

1. **Recommended Players**:
   - VLC Media Player (free)
   - Kodi (with PVR IPTV Simple Client addon)
   - IPTV Smarters (Windows version)
   - Network Player (NS player)

2. **VLC Media Player (easiest)**:
   - Open VLC
   - Menu → **Media** → **Open Network Stream**
   - Paste this link:  
     `https://raw.githubusercontent.com/johirxofficial/Toffee-Auto-Playlist/main/toffee_playlist.m3u`
   - Click **Play**
   - Channels list will appear on the left (Playlist panel)

3. **Kodi**:
   - Install **PVR IPTV Simple Client** addon
   - Settings → Add-ons → My add-ons → PVR clients → IPTV Simple Client → Configure
   - General tab → M3U Playlist URL → paste the m3u link above
   - Enable addon & restart Kodi
   - Go to TV section to see channels
